#elechelper

##About
Helper functions for selecting discrete electronic component values
Currently modelled:
- Resistors
- Capacitors
- Voltage
- Gain
- Frequency
- Wattage

Helpers:
- Parallel
- Normalize value
- R/C/F real value calculation
 
##Requirements
Python >= 2.6

##Installation
```python
python setup.py install
```

##Usage
View ```python
elechelper/examples``` folder
